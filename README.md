# GuuGoo

Go言語からRubyShellを呼び出すプログラム

```markdown

kaminarigumo.goを実行するとzinbeiのdオプションを呼びます。

環境構築は下を実行します。

gem install zinbei himekuri

※ WindowsでRubyInstaller2から、Cドライブ直下Ruby26-x64で入れていると仮定したPATHです。

```

※ Go言語でRubyShellを管理できるまで、進んでいきます。
